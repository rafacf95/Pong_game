# Pong Game

Uma implementação simples do clássico jogo **Pong**, criada com foco em
aprendizado usando Unity e C#.

## Sobre o projeto

Este projeto recria o jogo clássico *Pong*, permitindo ao jogador
enfrentar uma IA simples por meio do controle de paddles e uma bola com
física básica.\
O objetivo principal foi praticar lógica de jogo, manipulação de física
e estruturação de um projeto no Unity.

## Estrutura do repositório

    Pong_game/
    ├── Assets/            # Recursos do Unity (scripts, sprites, cenas, etc.)
    ├── ProjectSettings/   # Configurações do Unity
    └── (Arquivos padrão do Unity)

## Como executar

1.  Clone o repositório:

    ``` bash
    git clone https://github.com/rafacf95/Pong_game.git
    ```

2.  Abra o projeto no Unity (versão recomendada definida em
    `ProjectSettings/`).

3.  Abra a cena principal dentro da pasta `Assets/`.

4.  Clique em **Play** no editor para iniciar o jogo.

## Objetivos de aprendizado

-   Prática de lógica básica de jogos.
-   Movimentação de objetos usando física.
-   Controle de input do jogador.
-   Organização de projeto dentro da estrutura Unity.
-   Criação de mecânicas simples de IA.
